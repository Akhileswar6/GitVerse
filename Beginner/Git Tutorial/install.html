Git Getting Started

Install Git

You can download Git for free from the following website: git-scm.com
To start using Git, you need to install it on your system, Git is available for Windows, macOS, and Linux.

Windows

  • Go to the official Git website: git-scm.com
  • Click on Download for Windows.
  • Run the downloaded .exe file.
  • Choose default options unless you have specific needs.
  • After installation, open Git Bash, copy the following command and verify installation.
git --version
Result - Shows your current Git Version.

git version 2.47.1.windows.2

macOS

  • The easiest way is using Homebrew (if installed)
brew install git
  • Or install using Terminal by typing the following command
xcode-select --install
  • After installation, copy the following command and verify installation.
git --version
Result - Shows your current Git Version.

git version 2.9.5 (Apple Git-154)

Git Config

Before you start using Git, you need to set up your Git configuration. This includes setting your username and email address, which will be associated with your commits.

  • Replace "Your Name" with your username.
git config --global user.name "Your Name"
Example

git config --global user.name "GitVerse"

  • Replace "Your email" with your Email-id.
git config --global user.email "Your email"
Example

git config --global user.name "gitverse@gmail.com"


View your git configuration

  • Username
git config user.name
  • Email-id
git config user.email
